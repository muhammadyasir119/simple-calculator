# 🧮 Simple Calculator

A Python-based command-line calculator that performs basic math operations using functions and loops.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Continuous loop until user exits
- Error handling (division by zero)

---

## 🛠️ Technologies Used

- Python 3

---

## 📂 Project Files

- `calculator.py` → Basic calculator
- `calculator_advanced.py.py` → Advanced version
- `calculator_advanced_v2.py` → Improved version
